# TypedDB

A TypeScript wrapper for the IndexedDB browser API, enabling strongly typed ORM style querying.

## Setup

1. Install dependencies

    `npm install`

2. Build project

    `npm run build`

## Running tests

* **From a browser**

    `npm start`

    Navigate to `http://localhost:9000/__intern/`
